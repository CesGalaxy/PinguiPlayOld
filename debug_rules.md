# Debug Rules
## Console
### Colors
| Color   | Normal        | Bold      | Background |
|---------|---------------|-----------|------------|
| Cyan    | Debug         |           |
| Green   | Done!         |           |
| Yellow  | Warning       | Starting  | DANGER     |
| Red     | Error         |           | FATAL      |
| Magenta | IMPORTANT     |           |
| Gray    | Not important | Milestone | MILESTONE  |
| Blue    | Stats         |           | Updating   |
| White   | Misc          | LOOK      |
